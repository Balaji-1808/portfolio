# My Portfolio

Welcome to my personal portfolio project! This project showcases my skills, projects, and experiences in web development. It is built using modern web technologies and is fully responsive for all devices.

## Features

- **Responsive Design**: The portfolio is fully responsive and works seamlessly on all devices, including desktops, and mobile phones.
- **Interactive Header**: A sticky header with navigation links.
- **Project Showcase**: A section to display my projects with hover effects for better user interaction.
- **Modern Design**: Clean and modern UI with smooth transitions and animations.

## Technologies Used

- **Frontend**: React.js
- **Styling**: CSS (with responsive design)
- **Fonts**: Google Fonts (Lato, Poppins, Roboto)

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Balaji-1808/portfolio
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

5. Open your browser and navigate to `http://localhost:3000` to view the portfolio.

## Customization

- **Projects Section**: To add or update projects, modify the `Project` component and its associated CSS in `Project.jsx` and `Project.css`.

## Responsive Design

The portfolio is designed to be fully responsive. The `@media` queries in the CSS files ensure that the layout adapts to different screen sizes, including mobile devices.
